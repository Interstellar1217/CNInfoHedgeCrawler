# CNInfoHedgeCrawler

巨潮资讯网套期保值公告爬虫工具

## 项目简介

本项目用于从[巨潮资讯网](https://www.cninfo.com.cn)自动搜索和下载包含"套期保值"关键词的上市公司公告。支持按关键词搜索、日期过滤、自动翻页、断点续爬等功能。

## 功能特性

- ✅ 自动搜索"套期保值"相关公告
- ✅ 支持自定义关键词搜索
- ✅ 自动翻页获取全部结果
- ✅ 下载公告PDF文件到本地
- ✅ 保存公告元数据（标题、代码、日期等）到CSV
- ✅ 支持按日期范围筛选
- ✅ 断点续爬，避免重复下载
- ✅ 请求延时控制，避免对服务器造成压力
- ✅ 详细的日志记录
